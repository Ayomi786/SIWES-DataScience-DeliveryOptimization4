SIWES Data Science Project - Supply Chain & Delivery Optimization

This project demonstrates how to use data science to optimize supply chain routes for an e-commerce beverage company.
It includes clustering (K-Means), heuristic route optimization (Nearest Neighbor + 2-opt), and visualization.

Files included:
- SupplyChain_Delivery_Optimization.ipynb : Jupyter Notebook (full code and analysis)
- synthetic_orders.csv : Synthetic raw data
- synthetic_orders_with_clusters.csv : Clustered dataset
- optimized_routes_summary.csv : Optimized route summary
- route_cluster_*.png : Route visualization images

Developed for: SIWES Industrial Training Report
